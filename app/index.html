CSV Format Instructions:

For People (Humans), use these columns (in any order):

  • Name (required) - Full name
  • Birth Date (optional) - Format: YYYY-MM-DD
  • Notes (optional) - Additional information
  • Tags (optional) - Comma-separated tags
  • Friends (optional) - Comma-separated friend names
  • Family (optional) - Comma-separated family member names

For Companies/Organizations, use these columns:

  • Name (required) - Company/Organization name
  • Type (required) - e.g., Company, Organization, Government
  • Notes (optional) - Additional information

Note: The first row should be headers. Data will be automatically categorized based on the presence of 'Type' column.
